> Full-stack platform built on real government data from 1,663 Rae Bareli villages. >
> `Next.js` `FastAPI` `Gemini 2.0` `Groq Whisper` `Firestore` `Leaflet` `Cloud Run`

>
> `Next.js` `FastAPI` `ResNet50` `Groq` `PostgreSQL` `Prisma`

**[PitchPerfectAI](https://pitchperfectfrontend.vercel.app)** — AI public speaking coach
> Real-time facial expression + posture + speech analysis to coach public speaking. Face-API.js + MediaPipe Vision, Socket.io for live feedback, Recharts for session analytics.
>
> `React` `Face-API.js` `MediaPipe` `Socket.io` `Recharts`
